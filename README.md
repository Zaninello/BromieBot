# TaskBot
#### Este Bot do Telegram tem como função gerenciar tarefas interativamente com o usuário.

## :mag:Funcionalidades:  
[ ] Adicionar Tarefa  
[ ] Remover Tarefa  
[ ] Editar Tarefa  
[ ] Visualizar Tarefas  
[ ] Concluir Tarefa  

## :black_nib:Comandos:  
/Help 🠒 Catalogo de funções possíveis com os devidos comandos e normas de escrita. 
  
/Adicionar 🠒 Abre espaço para que seja adicionada uma tarefa, seguindo a norma de escrita aceita pelo Bot. 
  
/Remover 🠒 Espaço para citar qual task já criada deseja remover, seguindo a norma de escrita aceita pelo Bot.  
  
/Editar 🠒 Permite a edição de uma task já criada, seguindo a norma de escrita aceita pelo Bot.  
  
/Visualizar 🠒 Emite um catálogo de tarefas criadas pelo usuário solicitante, seguindo a norma de escrita aceita pelo Bot.  
  
/Concluir 🠒 Encerra a tarefa criada anteriormente, seguindo a norma de escrita aceita pelo Bot.  

## :tophat:Como Acessar o Bot:
Procure por "" no Telegram e inicie o chat enviando-o uma mensagem agradável.
